# Springboot2+Quartz+MybatisPlus+easymock
通过easymock模拟数据，使用restTemplate去获取这些mock数据，结合Jackson将json转换成实体类，利用 MybatisPlus 批量存储数据， 通过 Quartz 实现一个定时 job ，重复执行上面的步骤。

帮助小伙伴们快速熟悉这个MP的使用和了解下Easymock😄
